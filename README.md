## 技术细节
国家地理的UI基本都是基于原生实现     
使用了Android Architecture Components + Databinding作为整体架构        
使用了Kotlin

使用了以下开源库

* Retrofit
* OkHttp
* Glide
* PhotoView


国家地理的技术

* 手动实现了上拉加载更多，并加入了加载错误的容错，点击重新加载回调
* CardView + 多种type的RecyclerView展示列表
* SQLite实现本地收藏
* 使用ViewPager来展示图片
* OkHttp + Retrofit实现的网络模块
* Glide加载图片，下载图片，处理图片缓存
* webview显示页面
* 简单的分享功能
